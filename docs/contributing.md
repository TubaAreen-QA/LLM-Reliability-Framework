# Contributing

## Development

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run tests

```bash
pytest
```

---

## Coding Standards

- Python 3.11+
- Type hints required
- Black formatting
- Ruff linting
- Pytest for tests

---

## Pull Requests

Every pull request should

- Pass CI
- Include tests
- Update documentation
- Keep backward compatibility
